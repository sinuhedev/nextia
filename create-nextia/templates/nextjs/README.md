# TEMPLATE

## To start

Open http://localhost:3000 to view it in the browser.

```sh
npm i
npm start
node --run start
#
npm test
npm run build
npm run serve
```

## env

```.env
.env              # loaded in all cases
.env.[ENV]    # only loaded in specified ENV [ development, test, production ]
```

 * .env.production
 * .env.test

NEXT_PUBLIC_LOGGER=true
